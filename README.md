# PeopleManager

A simple Node.js terminal application to manage people data using CRUD operations.

## Features

- Add data for 10 people
- View all people data
- View a specific person by id
- Delete all people
- Delete a specific person
- Display full name and city
- Input validation using validator.js

## Technologies Used

- Node.js
- readline-sync
- validator
